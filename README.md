# Васькина Яна ИТС-2 Лабораторная №2

# Задание 1

### Текст задачи

На основе списка, содержащего цифры от 1 до 9, получить список из их римских обозначений.

### Алгоритм решения

1. Запрашиваем у пользователя числа от 1 до 9
2. Создаем рекурсивную функцию read с параметром acc, для ввода данных. Проверяем, можно ли преобразовать введенные значения в числа
3. Создаем список list, который запускает read с пустым списком, и переворачиваем список с помощью List.rev, чтобы числа шли в порядке ввода
4. Создаем функцию Roman для преобразования цифр в римские, сопоставляем цифру со значением
5. Преобразуем список с помощью list.map, применяя функцию к каждому элементу списка
6. Выводим результат

# Задание 2

### Текст задачи

Список содержит десятичные цифры. Составить из них число.

### Алгоритм решения

1. Запрашиваем у пользователя десятичные цифры (от 0 до 9)
2. Создаём рекурсивную функцию read с параметром acc, которая считывает ввод пользователя. Проверяем, можно ли преобразовать введённое значение в число и находится ли оно в диапазоне от 0 до 9. Если да — добавляем цифру в список и продолжаем ввод
3. Создаём список digits, который запускает функцию read с пустым списком, после чего переворачиваем список с помощью List.rev, чтобы цифры шли в порядке ввода
4. Создаём значение number, где из списка цифр формируется одно число
5. Используем List.fold, чтобы пройти по списку цифр и на каждом шаге умножать текущее число на 10 и прибавлять следующую цифру
6. Выводим список введённых цифр
7. Выводим полученное число, составленное из этих цифр
